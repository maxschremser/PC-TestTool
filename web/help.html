PC-TestTool Hilfe

Downloaden der Anwendung

Sie können das PC-TestTool auf verschiedene Arten starten:
JavaWebStart
  1. Öffnen Sie dazu die Webseite http://maxi.schremser.com/Content.Node2/PC-TestTool/, klicken Sie auf den Link Start PC-TestTool und bestätigen Sie mit OK.

    2. Sollten Sie bis jetzt noch kein Java installiert haben, laden Sie Java von der Webseite http://www.oracle.com/technetwork/java/javase/downloads/index.html herunter. Sie müssen sich gegebenenfalls registrieren bevor Sie mit dem Download fortsetzen können.

    Laden Sie die letzte Version von Java SE durch anklicken herunter
    Folgen Sie bitte den Anweisungen auf der Webseite.

  2. Zuerst kommt ein Startbild von Java WebStart
    .
  3. Nachdem Java WebStart gestartet ist, wird die Antwendung aus dem Internet heruntergeladen. Warten Sie bis die Anwendung komplett heruntergeladen wurde.
  4. Nachdem Sie Anwendung installiert wurde, müssen Sie nur noch den Link aufrufen um die Anwendung zu starten. Dies geht dann im Allgemeinen schneller, da die notwendigen Dateien nicht mehr erneut heruntergeladen werden müssen.
ZIP
  1. Klicken Sie auf den Link Download TestTool-1.3.zip um die Anwendung als ZIP komprimierten Ordner herunterzuladen.
  2. Speichern Sie die ZIP Datei zum Beispiel auf dem Desktop.
  3. Klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie Extract All.... Klicken Sie auf den Extract Button.
  4. Entpacken Sie die beinhalteten Dateien.
  5. Doppelklicken Sie anschließend auf die Datei run.bat und bestätigen Sie mit Run.
  6. Sofern Java bereits installiert ist startet die Anwendung automatisch.

Benützung der Anwendung

  1. Wenn Sie die Anwendung erfolgreich gestartet haben, müssen Sie auswählen was für eine Art Test sie durchführen wollen
    • Österreichische Fischereigesellschaft
    • Wiener Fischereiausschuss

  2. Klicken Sie auf das entsprechende Bild um fortzufahren.
  3. Sie haben jetzt im Prinzip zwei Möglichkeiten eine Prüfung zu erstellen
    1. Durch manuelle Eingabe
    2. Durch Importieren einer Excel Datei
